## 📊 儀表板功能

### V2 新增功能
- **出租狀態切換**：點擊按鈕即時切換各房間、套房、停車位的出租狀態
- **月收入 / 年收入**：自動加總目前出租中的租金
- **稅務試算**（點擊「展開明細」）
  - 🏠 房屋稅：評定現值 × 路段率 × 稅率（誠實申報出租 1.5%–2.4%）
  - 🌱 地價稅：公告地價 × 持有坪數 × 累進稅率
  - 📋 租賃所得稅：年租金 × (1 − 43% 必要費用) × 綜所稅率
- **稅後利潤**：年收入 − 總稅額，附留存率顯示
- **PARAMS 參數面板**：可手動輸入實際評定現值、公告地價、稅率，即時重算

### 使用方式

```
直接用瀏覽器開啟 rental_dashboard_v2.html，無需 server 或安裝任何套件。
```

## 🧮 稅務計算邏輯

依據 `land_estate.json` 中定義的公式與 2026 年囤房稅 2.0 政策：

### 房屋稅
```
房屋稅 = 房屋評定現值 × 路段率 × 稅率
```
- 誠實申報租賃所得：1.5% – 2.4%
- 非自住未出租：最高 4.8%（雙北 7 戶以上）

### 地價稅
```
地價稅 = 公告地價（元/坪）× 持有坪數 × 稅率
```
- 自用住宅用地：0.2%
- 一般累進：1.0% – 5.5%

### 租賃所得稅
```
應納稅額 = 年租金收入 × (1 − 43%) × 綜所稅率
```
- 43% 為法定必要費用扣除率
- 綜所稅率依個人級距（預設 20%，可在 PARAMS 調整）

> ⚠️ 以上為估算參考。評定現值與公告地價請以稅務機關實際核定數字為準。
